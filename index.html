Read the full breakdown

An 80+ person global PMO had grown through title-driven org design rather than work-driven design. I led a capability assessment using AI-assisted analysis of de-identified task data against ITIL and PMBOK standards. The finding: fewer than 10% held PMP certification despite formal PM titling, with people in identical roles performing fundamentally different work.

Built a Task → Capability → Role → Organization model — career ladders, competency matrices, a certification roadmap and policy, KPI and performance frameworks, and a standardized governance model — giving leadership an objective, fact-based foundation for workforce planning instead of an org chart built on assumption.
